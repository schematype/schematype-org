SchemaType Resources

Need help with SchemaType? Try these resources.

Twitter

Follow @SchemaType on Twitter.

Wiki

Our wiki contains information and conversations about all the things you might want to do, or problems you might run into with SchemaType.

IRC

Get live support at #schematype on irc.freenode.net, the official SchemaType IRC channel.

Slack

If you prefer Slack we have that too. The Slack and IRC channels are interconnected by a bot. To get an invite to our Slack channel, just click on the IRC channel link above and ask for one.

Issues

Search through the issues on the main SchemaType development. Think you’ve found a bug? File a new issue.
