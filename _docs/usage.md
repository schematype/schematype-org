The SchemaType NPM package installs a `schematype` executable available to you
in your Terminal window. You can use this command in a number of ways:

{% highlight bash %}
$ schematype validate
# => The ...

$ schematype generate
# => The ...

$ schematype translate
# => The ...
{% endhighlight %}
